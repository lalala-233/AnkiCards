# Anki Cards

这是一个存储着 Anki 数据的备份。

## 特点

纯文本存储，使用 Git 进行备份。

## 目录

### [New](/New/)

[New](/New/) 目录下存储的是现在正在使用的一些数据。

数据格式为 [AnkiMaker](https://github.com/lalala-233/AnkiMaker) 可识别的 `toml` 文件，以及生成的可导入的 `txt` 文件。

### [Old](/Old/)

[Old](/Old/) 目录下存储的是一些以前的数据，仅作备份。
